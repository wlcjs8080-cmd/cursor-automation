# PROJECT PLAN - 문서 자동화 시스템 (2026-02-24 업데이트)

## 한 줄 요약
스케줄 시트에 업무를 기록하면, 자동으로 레포트 파일을 만들고 마스터 시트에서 필요한 정보를 가져와 채워주는 프로그램

## 전체 Step

### Step 0: 초기화
- 자동화 이전에 수동 처리한 건에 AC열 기존완료 표시
- 1회만 실행
- 상태: 완료

### Step 1: 레포트 자동 생성
- 스케줄 시트에서 AB열(방문완료)=O + AC열(처리완료)=비어있는 행을 찾음
- 3월 블록(HEADER_ROWS 183)부터만 실행
- 스케줄 데이터를 레포트 템플릿에 채움
- 마스터에서 SET UP(V10), 이전 점검일(V11) 가져옴
- 고객사 > REPORT 폴더에 파일 저장
- AC열에 완료 표시
- 상태: 테스트 성공 (2026-02-24)

### Step 2: Part별 이전 교체일 (K57~K63)
- 이전 점검일(V11)은 Step 1에서 처리 완료 -> Step 2에서 제외
- Step 2는 K57~K63(Part별 이전 교체일)만 처리
- 실행 방식: 사람이 레포트를 열어둔 상태에서 앱의 Step 2 버튼 클릭
- xlwings가 열려있는 레포트에서 직접 읽기/쓰기 (저장 불필요, 화면 즉시 반영)
- 동작 순서:
  1) 사람이 레포트 파일을 열고 I57~I63에 품번 수동 입력
  2) 앱에서 Step 2 버튼 클릭
  3) 코드가 열려있는 레포트에서 N11(S/N)과 I57~I63(품번) 읽음
  4) New Alarm 및 사용Part 이력에서 S/N(L열)+품번(AC열) 매칭
  5) 오늘 이전 가장 최근 작업일자(P열) 가져옴
  6) K57~K63 각 셀에 이전 교체일 바로 입력
  7) 사람이 화면에서 결과 확인 후 저장
  8) I57~I63에 품번 비어있으면 해당 행 건너뜀
- 상태: 미착수

### Step 3: 마스터 시트에 데이터 기입
- 스케줄 시트 + 레포트 시트 내용을 마스터 시트에 새 행으로 기입
- 상세 내용 미정 - 착수 전 DK님과 확정 필요:
  - 스케줄 시트에서 가져올 항목과 마스터 시트 대상 열
  - 레포트 시트에서 가져올 항목과 마스터 시트 대상 열
  - 기입 대상 시트: New Alarm 및 사용Part 이력이 맞는지
  - 새 행 추가 위치 (마지막 행 아래? 특정 위치?)
  - 트리거 조건 (레포트 저장 시 자동? 앱에서 Step 3 버튼?)
- 상태: 미착수

### 최종: .exe 앱 변환
- 버튼으로 각 Step 실행 (Step 1, Step 2, Step 3 버튼)
- 서버 경로 설정 가능
- 5명 각자 컴퓨터에서 사용 가능
- 상태: 미착수

## 시트 구조 (실제 확인 완료)

### 스케줄 시트 (Sheet1)
- HEADER_ROWS = [93, 138, 183, 229, 274, 319, 364]
- 3월(183)부터 실행
- 열: J=접수일자, K=방문계획, L=1차방문, M=2차방문, N=3차방문, O=미준수사유, P=고객사, Q=담당자, R=접수자, S=MAT작업자, T=업무내용, U=유무상, V=인원, W=공정, X=라인, Y=MODEL, Z=설비호기, AA=S/N, AB=방문완료, AC=처리완료

### 설비 Master Sheet
- S/N=J열(col10), SET UP=M열(col13), 머릿말 6행, 데이터 7행부터

### New Alarm 및 사용Part 이력
- S/N=L열(col12), 품번=AC열(col29), 작업일자=P열(col16), SET UP=AL열(col38), 머릿말 2행, 데이터 3행부터

### 레포트 템플릿 (PM REPORT 새양식)
- E10=고객사, E11=담당자, E12=MAT작업자, N10=MODEL, N11=S/N, N12=공정
- V8=점검일, V10=SET UP, V11=이전 점검일, 경과일=엑셀수식
- I57~I63=품번(수동입력), K57~K63=이전교체일(Step2)

## SET UP 조회 로직
- 1순위: 설비 Master Sheet S/N(J열) -> SET UP(M열)
- 2순위: New Alarm 시트 S/N(L열) -> SET UP(AL열)
- 둘 다 없으면 오류
