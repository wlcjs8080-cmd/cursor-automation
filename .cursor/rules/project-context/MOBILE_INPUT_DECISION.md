# 모바일 입력 방식 결정 (미확정 - 현장 인터넷 확인 후 결정)

## 결정 기준: 현장 인터넷 안정성

## 선택지 비교

| 항목 | 커스텀 앱 | Google Forms | Microsoft Power Apps |
|------|----------|-------------|---------------------|
| 개발 난이도 | 매우 높음 | 매우 낮음 | 중간 |
| 개발 기간 | 수 주~수 개월 | 30분~2시간 | 1~2주 |
| 비용 | 무료 (직접 개발) | 무료 | 월 $160 (8명) |
| 오프라인 지원 | 가능 | 불가 | 제한적 |
| UI 자유도 | 최고 | 낮음 | 높음 |
| 입력 정확성 | 최고 | 보통 | 높음 |
| 기존 자동화 연동 | 직접 코딩 | API로 쉬움 | 설정 복잡 |
| 확장성 | 무제한 | 보통 | 높음 |
| 유지보수 | 직접 해야 함 | Google이 관리 | Microsoft가 관리 |

## 판단 기준
- 인터넷 안정 → Google Forms (최우선 추천)
- 인터넷 불안정 → 커스텀 앱 (유일한 오프라인)
- 절충 → Google Forms 먼저, 필요시 커스텀 앱 전환

## 확정 후 다음 단계
1. 선택한 방식의 구현 코드 작성
2. 기존 자동화(Step 1~3)와 연동 설계
3. DB 마이그레이션과 병행 진행

## 상태: ⏳ 현장 인터넷 확인 대기 중
